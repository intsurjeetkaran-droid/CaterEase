🧾 8. LOGGING SYSTEM (IMPLEMENTED — Winston)
Auth    → logger.info('User login', { user_id })
         logger.info('User registered', { user_id, role })
Order   → logger.info('Order created', { order_id, total_amount })
         logger.info('Order status updated', { order_id, status })
Payment → logger.info('Payment added', { order_id, amount })
Error   → logger.error('Error occurred', { message, stack })

Log files:
  backend/logs/combined.log   ← all logs
  backend/logs/error.log      ← errors only

 ├── app.js                  ← Express entry point
